# Personal Website

Static personal website built with Astro, Tailwind CSS, content collections, and React icons.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Content Editing

Update site content without touching layout code:

- `src/content/publications/`
- `src/content/projects/`
- `src/content/blog/`

The site deploys to GitHub Pages through `.github/workflows/deploy.yml`.
